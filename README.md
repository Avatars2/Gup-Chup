# Gup-Chup

A real-time chat app built with React on the frontend and Node.js/Express on the backend.

## Features

- User authentication and profiles
- Real-time messaging with WebSockets
- File uploads for images, videos, audio, and documents
- Group chat support
- Message editing and deletion
- Responsive dark-themed UI

## Project Structure

- `backend/` - Express server, routes, controllers, models, middleware
- `frontend/` - React app with Tailwind CSS and chat UI components

## Setup

1. Install backend dependencies:
   ```bash
   cd backend
   npm install
   ```

2. Install frontend dependencies:
   ```bash
   cd ../frontend
   npm install
   ```

3. Run backend and frontend in separate terminals.
